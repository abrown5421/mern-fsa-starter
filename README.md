MERN Feature-Sliced Starter

A modern full-stack MERN boilerplate using a monorepo setup and Feature-Sliced Architecture (FSA) on the frontend.
Designed to scale from small projects to large applications while maintaining clear boundaries and predictable structure.

Features:
- Feature-Sliced Architecture (FSA) for scalable frontend code
- Monorepo structure (frontend + backend in one repo)
- React + TypeScript
- Node.js + Express + MongoDB
- Ready for auth, API expansion, and shared types
- Clean separation of concerns
- Ideal starting point for real-world apps

Prerequisites:
Before starting, make sure you have MongoDB installed and running on your system, as this project requires a MongoDB database.

Getting started:
To create your application, utilize degit to clone the repository with your given project name, install dependencies, run the unit tests, and start the application.

```
degit abrown5421/mern-fsa-starter my-project
cd my-project
pnpm install
pnpm test
pnpm dev
```
